# Computer Vision Portfolio - Padmanabh Butala

A futuristic, interactive portfolio website showcasing expertise in Computer Vision, Machine Learning, and AI. Built with a "Vision Processing Pipeline" theme that reflects the journey from raw data to intelligent insights.

## 🎯 Overview

This portfolio represents the complete computer vision development lifecycle through its design and functionality:

- **Hero Section (Image Acquisition)**: Point cloud visualization with real-time particle interactions
- **About Section (Preprocessing Pipeline)**: Feature detection animations and professional timeline
- **Skills Section (Neural Network Architecture)**: Interactive neural network with animated connections
- **Projects Section (Model Inference)**: Production-ready CV systems with performance metrics
- **Contact Section (Output & Deployment)**: Terminal-style interface for professional connections

## 🚀 Features

### Visual Design
- **Color Palette**: Neural network inspired with deep blues, electric accents, and activation highlights
- **Typography**: Technical fonts (JetBrains Mono, Space Grotesk) for authentic developer feel
- **Animations**: CV-specific effects including SIFT-like feature detection, attention maps, and data flow visualizations

### Interactive Elements
- **Point Cloud Background**: Real-time particle system with mouse interaction
- **Neural Network Visualization**: Animated connections between skill layers
- **Attention Maps**: Mouse-following effects on interactive elements
- **Feature Detection**: Animated keypoints on profile image
- **Terminal Interface**: Typewriter effects and command-line styling

### Technical Implementation
- **Performance Optimized**: 60fps animations with GPU acceleration
- **Responsive Design**: Mobile-first approach with touch-friendly interactions
- **Accessibility**: ARIA labels, semantic HTML, and reduced motion support
- **Modern JavaScript**: ES6+ classes, Intersection Observer API, and performance monitoring

## 🛠️ Technologies Used

### Frontend
- **HTML5**: Semantic structure with accessibility features
- **CSS3**: Advanced animations, Grid, Flexbox, and custom properties
- **JavaScript (ES6+)**: Modern syntax with classes and modules

### Key Libraries & APIs
- **Canvas API**: Point cloud particle system
- **Intersection Observer**: Scroll-triggered animations
- **Web Animations API**: Performance-optimized transitions
- **SVG**: Neural network connection lines

### Performance Features
- **Lazy Loading**: Images and resources loaded on demand
- **Debounced Events**: Optimized scroll and resize handlers
- **GPU Acceleration**: Transform-based animations
- **Reduced Motion**: Accessibility compliance

## 🎨 Design Philosophy

### Computer Vision Theme
The website mimics a computer vision processing pipeline:

1. **Data Acquisition**: Hero section with point cloud visualization
2. **Preprocessing**: About section with feature extraction effects
3. **Feature Learning**: Skills displayed as neural network layers
4. **Model Inference**: Projects with performance metrics
5. **Output Generation**: Contact section as system deployment

### Color Psychology
- **Neural Blue (#1a1a2e)**: Deep learning and neural networks
- **Processing Blue (#16213e)**: Data transformation stages
- **Activation Orange (#e94560)**: Neural activation and highlights
- **Data Flow Cyan (#0f4c75)**: Information pathways
- **Output White (#eee)**: Final results and clarity
- **Accent Green (#64ffda)**: Success states and interactions

## 🔧 Setup & Installation

### Local Development
```bash

git clone https://github.com/Padmanabh03/portfolio.git

# Navigate to project directory
cd portfolio

# Open in browser (no build process required)
open index.html
```

### Live Server (Recommended)
```bash

python -m http.server 8000

npx http-server
```
